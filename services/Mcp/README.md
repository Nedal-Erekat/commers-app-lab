# MCP Server

Planned in **milestone 6** (see [ROADMAP.md](../../ROADMAP.md)).

.NET MCP server exposing commerce tools (`search-products`, `get-order-status`, `recommend-products`, `add-to-cart`) backed by the Catalog/Order services. Consumed by the milestone 7 AI shopping assistant as an MCP client, and usable by any other MCP client (Claude, Copilot, etc.).
